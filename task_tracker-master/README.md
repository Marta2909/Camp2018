# README

## Setup

### Database:

```bash
bundle install
rails db:migrate
```

### Running the app:
```bash
rails s
```

### Running tests:
```bash
bundle exec rspec
```

### Running rubocop linter:
```bash
bundle exec rubocop
```

### Running tests and liters automatically with guard:
```bash
bundle exec guard
```
